# Skin Lesion Segmentation

Notebook project for medical image segmentation using the PH2 skin lesion dataset.

## What is inside

- dataset preparation
- training and validation pipelines
- `SegNet` implementation
- `UNet` experiments
- comparison of `BCE`, `Dice`, and `Focal Loss`

## Metric

`IoU`

- `skin_lesion_segmentation.ipynb`
