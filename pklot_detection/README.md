# PKLot Parking Space Detection

Notebook project for parking space detection and occupancy recognition using the PKLot dataset and Ultralytics YOLO.

## What is inside

- conversion from COCO annotations to YOLO format
- dataset preparation for Kaggle runtime
- training and validation of a YOLO model
- saved model weights and training artifacts
- evaluation with confusion matrices and summary metrics

## Metrics

- `Precision`
- `Recall`
- `mAP@50`
- `mAP@50-95`

## Best reported validation results

- Precision: `0.9982`
- Recall: `0.9917`
- mAP@50: `0.9928`
- mAP@50-95: `0.9684`
